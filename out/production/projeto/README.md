#   CALCULADORA EM JAVA

## Descrição

A calculadora foi feita em java, para mostrar os conceitos básicos de programação, como variáveis, operadores, laços de repetição, estruturas condicionais, operadores lógicos, entre outros.
Foram utilizados conceitos de orientação a objetos, como classes, métodos e objetos, encapsulamento, herança, acoplamento entre outros.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Potência

##  Conceitos Aplicados

- **Encapsulamento**: Uso de modificadores privados e métodos getters/setters.
- **Associação**: A classe de teste utiliza um objeto da classe Calculadora.
- **Estruturas de Controle**: Loop `while` para manter o programa rodando e `switch case` para o menu.

## Construção

- Primeiramente, foi criado um projeto no Intellij, com o nome de "Calculadora".
- Após isso, foi criado um pacote chamado "Domain" e um pacote chamado "Test", dentro do projeto.
- Dentro do pacote "Domain", foi criado uma classe chamada "Calculadora", que contém os métodos de soma, subtração, multiplicação e divisão.
- Dentro do pacote "Test", foi criado uma classe chamada "CalculadoraTest", que contém os testes na classe de execução para os métodos de soma, subtração, multiplicação e divisão.

## Como executar 

- Baixe o compilador do intellij IDEA: https://www.jetbrains.com/idea/download/
- Use o git hub desktop para baixar o projeto na raiz do seu computador.
- Abra o projeto no intellij IDEA.
- Clique no botão "Run" na barra de ferramentas do intellij IDEA.
- A calculadora será executada e você poderá usar os métodos de soma, subtração, multiplicação e divisão e potenciação.
